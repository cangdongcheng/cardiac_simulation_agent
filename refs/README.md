# refs/

The **EXTERNAL**: pointers to papers, manuals, repositories, and external resources.

A note belongs here if it's a curated index of external material — not original content.

## Structure

Single notes here are usually annotated bibliographies or reading lists. One note per topic-cluster, not one per paper.

External references that get repeatedly cited inline in other notes (Land 2017, Holzapfel-Ogden 2009, Niederer 2011, etc.) belong in this dir's bibliography file once it exists.
